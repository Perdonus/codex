# Codex Android Client Repo Instructions

- This repository is the Android client for remote `codex-cli`.
- Do not run Android builds on this Linux server.
- Build and release only through GitHub Actions.
- The canonical GitHub remote for this repo is `git@github.com:Perdonus/codex.git`.
- When changing CI, keep the workflow GitHub-only and avoid any server-side APK/AAB build steps.
